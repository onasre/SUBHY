# SUBHY
A comprehensive desktop (Windows) application designed as a digital archive for the intellectual works, books, and articles of Islamic thinker Dr. Ahmed Subhy Mansour. The application provides offline access to his entire literary catalogue, featuring advanced text search capabilities and an optimized reading interface

## 🌍 نظرة عامة | Overview

**بالعربية:**  
تطبيق مكتبي (ويندوز) يمثل أرشيفاً شاملاً ورقمياً لأعمال ومؤلفات المفكر الدكتور أحمد صبحي منصور. يهدف التطبيق إلى تيسير الوصول إلى كامل نتاجه الفكري من مقالات، كتب، وأبحاث منشورة، مع توفير ميزات متقدمة للبحث، القراءة السلسة، والتصفح دون الحاجة للاتصال بالإنترنت.

**In English:**  
A comprehensive desktop (Windows) application designed as a digital archive for the intellectual works, books, and articles of Islamic thinker Dr. Ahmed Subhy Mansour. The application provides offline access to his entire literary catalogue, featuring advanced text search capabilities and an optimized reading interface.

## ✨ المميزات الرئيسية | Key Features

* 📚 **محتوى شامل / Comprehensive Archive:** يضم كافة الكتب، الفتاوى، والمقالات الفكرية الخاصة بمنهج أهل القرآن.
* 🔍 **محرك بحث ذكي / Advanced Search:** إمكانية البحث السريع بالكلمات المفتاحية داخل النصوص والعناوين.
* 🌐 **تصفح دون إنترنت / 100% Offline:** الوصول إلى الأرشيف في أي وقت دون اشتراط وجود اتصال شبكة.
* 🖥️ **واجهة مريحة / User-Friendly UI:** تصميم مخصص لتسهيل القراءة الطويلة والاطلاع على الأبحاث بشكل سلس.
